# Personal Finance Tracker (Python)

A simple command-line application that allows users to record expenses,
categorize spending, and save data to a CSV file. Built using Python 
with a focus on clean code, modular functions, and user-friendly interaction.

## Features
- Add new expenses with date, category, and amount
- Automatically saves entries to data.csv
- View all expense records
- Filter expenses by month (coming soon)
- Summary statistics (coming soon)

## Technologies Used
- Python (File handling, datetime, CSV)
- Modular programming structure

## How to Run
1. Clone this repository:
   git clone https://github.com/nsabit/finance-tracker-python

2. Run the script:
   python main.py
